ARE YOU SPENDING TOO MUCH TIME ON CASE FINDING?

We can help

Case Finder is an online tool that suggests histologies and primary sites for uploaded pathology cases. Users can review the suggestions, accept them for import into Metriq, or discard them as not reportable. Reviewed cases can be exported into a file that will then be used to import the cases into Metriq as suspense cases.

Request A Demo

Key Features

Automated pull of data

Case Finder ingests order and result HL7 messages for pathology cases via HTTP from a local EHR.

NLP processing

Case Finder uses advanced natural language processing (NLP) techniques to analyze the text of the pathology case to generate suggestions for ICD-O-3 histology and sites - the data points necessary to determine if a pathology case contains a reportable cancer diagnosis.

Intuitive UI for expert review

Case Finder presents the suggested sites and histologies for pathology cases that have a suggested ICD-0-3 histology to the user for confirmation or revision and provides visual highlighting for the substantiation of its NLP suggestions.

Exportable data

Case Finder allows the user to submit for export any pathology cases with a confirmed site/histology pairing.
